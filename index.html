About Me

I am a Control Engineering graduate with a strong interest in UI/UX design and front-end development. I enjoy transforming ideas and user needs into intuitive, functional, and visually clean digital interfaces.

My background in engineering helps me approach design problems analytically. I work on user research, wireframing, and prototyping using tools such as Figma and Adobe XD, while also implementing designs using HTML, CSS and Java Script to ensure feasibility and consistency between design and development.

Skills

User Research

I collect and analyze user data to understand user needs and behaviors. I use interviews and surveys to extract insights that guide design decisions.

Wireframing

I create low-fidelity screen layouts to define content structure and user flow. I focus on clarity and usability before moving to visual design.

Prototyping

I build interactive prototypes to simulate the user experience. I use them to test ideas and gather early user feedback.

HTML & CSS

I develop responsive and accessible web interfaces. I focus on clean code, performance, and consistent styling.

Java Script

I add interactivity and dynamic behavior to user interfaces. I enhance usability through events, validations, and DOM manipulation.
